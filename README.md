Before downloading the data, you need to download your Kaggle API key.

* **Log in** to your Kaggle account.
* Go to the **Settings** page (Click your profile picture -> Settings).
* Scroll down to the **API** section.
* Click the **"Create New Token"** button. This will automatically download a file named `kaggle.json` to your computer.
